link do aplikacji:
https://damkol14.github.io/Ip-Address-Tracker/

Jest to aplikacja, która po wpisaniu ip lub nazwy strony 
pokazuje dane firmy, i znajduje jej lokalizację na mapie.

Do jej stworzenia użyłem:
- html,
- css,
- javascript,
- bibliotekę axios(do połączenia z api),
- IP Geolocation API by IPify(api zawierające dane firm, które można wyszukać np. po ip),
- LeafletJS(biblioteka umożliwiająca wygenerowanie mapy)

Pomysł na projekt zaczerpnięty ze strony frontendmentor.io
link https://www.frontendmentor.io/challenges/ip-address-tracker-I8-0yYAH0

Cały kod napisałem sam. Strona zapewnia podstawowe informacje, jak ma działać
aplikacja, użyte kolory, czcionki, potrzebne obrazy i ikony.
